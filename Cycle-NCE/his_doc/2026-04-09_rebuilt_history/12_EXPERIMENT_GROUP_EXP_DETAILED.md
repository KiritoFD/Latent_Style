# 实验组专档：EXP 主线组

数据源：`Y:\experiments\EXPERIMENT_RECORD_FULL_DATA.csv`

## 1. 组定位

主线实验与对照组（EXP）一共 31 条，是整个实验体系的中轴线。

它回答的不是“某个模块是否必要”，而是：

- 在当前主架构不彻底改写的情况下，主开关怎么影响 style/content 平衡。

## 2. 关键实验

### 2.1 `exp_1_control`

- style = `0.6502712479`
- lpips = `0.3731396071`
- fid = `290.7709695`
- art_fid = `405.2762803`

意义：

- 作为主线中的稳定参考点，说明没有极端开关时的均衡水平。

### 2.2 `exp_2_zero_id`

- style = `0.6509758176`

- fid = `289.1622874`
- art_fid = `412.3416869`

意义：

### 2.3 `exp_3_macro_strokes`

- style = `0.6480084022`
- lpips = `0.3578462195`

- 旧分析把它看作“均衡型”的好版本之一。

### 2.4 `exp_5_signal_overdrive`

- lpips = `0.3927676343`

- art_fid = `421.8789311`

### 2.5 `exp_S1_zero_id`

- 可以作为后续所有正式文档里反复引用的反例。

### 2.6 `exp_S2_color_blind`

- style = `0.7022627409`

- fid = `328.8214143`
- art_fid = `524.0720579`

意义：

### 2.7 `final_demodulation`

- style = `0.6986483848`
- lpips = `0.7188968239`
- fid = `456.8166908`
- art_fid = `798.8749670`

意义：

- 说明 demodulation 方向并没有给出真正可用的最终解。

## 3. 这组实验支持了哪些大结论

1. 不能只看 style。
2. identity 不能简单归零。
3. color 相关约束是必要项。
4. 主线最优往往是“均衡点”，不是单指标冠军。
